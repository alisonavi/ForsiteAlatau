# Alatau City Bank — Информационно-аналитический комплекс

Интерактивный аналитический комплекс банка **Alatau City Bank**, построенный на **Next.js 14 (App Router)** + **TypeScript** + **Tailwind CSS** + **Apache ECharts**.

Приложение переосмысливает концепцию форсайт-платформы (главная страница, радар, аналитика) под банковскую предметную область. **Бэкенд не требуется** — все данные моделируются на стороне клиента детерминированно (seeded), фильтры пересчитывают графики в браузере.

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
```

Прод-сборка:

```bash
npm run build && npm run start
```

## Структура

- `app/page.tsx` — **Главная**: KPI, интерактивная карта филиальной сети по регионам Казахстана, навигация.
- `app/radar/page.tsx` — **Форсайт-радар**: продуктово-сегментная матрица (BCG), стратегический профиль (spider), прогноз активов.
- `app/analytics/*` — 9 разделов аналитики, у каждого — **свой набор графиков**:
  | Раздел | Ключевые графики |
  |---|---|
  | Финансовые показатели | комбинированный, накопительная область, кольцевая, водопад (P&L bridge) |
  | Банковские портфели | treemap, 100% stacked, комбо+NPL, sunburst |
  | Риски | gauge (CAR), тепловая карта рисков, мульти-линия, стек по отраслям |
  | Мобильное приложение | воронка онбординга, DAU/MAU, оценки, платформы |
  | Продукты | «роза», sankey кросс-продаж, группированный бар, проникновение |
  | Маркетинг | themeRiver расходов, ROI-пузыри, расходы по каналам, CAC/LTV |
  | KPI | радиальные кольца, факт/цель (bullet), динамика, статус целей |
  | ФОТ и штат | возрастно-гендерная пирамида, ФОТ по департаментам, динамика, филиалы |
  | Налоги по филиалам | карта Казахстана, структура налогов, ТОП-регионы, динамика |

## Ключевые модули

- `components/map/KazakhstanMapSVG.tsx` — векторная карта регионов РК (порт из исходного приложения).
- `components/map/BranchesMap.tsx` — интерактивная хороплет-карта с тултипами.
- `components/ui/Chart.tsx` — обёртка ECharts (client-only).
- `lib/chart.ts` — единая палитра и стилевые токены графиков.
- `lib/rng.ts` — детерминированные генераторы данных (seeded).
- `lib/regions.ts` — метрики по регионам/филиалам.

> Демо-режим: цифры смоделированы для демонстрации и не являются реальной отчётностью банка.
